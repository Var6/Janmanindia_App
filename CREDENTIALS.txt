HOW TO USE
----------
1. Start dev server:   npm run dev
2. Open browser:       http://localhost:3000/dev
3. Click any role button to enter that dashboard instantly
   (no database required — uses DEV_BYPASS cookie)

on the /dev page (requires MONGODB_URI to be set):

NOTES
-----
- DEV_BYPASS=true is set in .env.local — all dashboards render
  with empty state if MONGODB_URI is not configured.
- To use real data: add MONGODB_URI to .env.local and seed users
  from the /dev panel.
- Remove DEV_BYPASS from production environment variables.
- JWT_SECRET in .env.local is for development only — set a strong
  random value in production.
